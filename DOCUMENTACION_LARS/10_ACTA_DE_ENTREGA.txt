2. OBJETO DE LA ENTREGA
================================================================================

Por medio del presente documento se hace entrega formal del Sistema de Gestión
Documental KAPA, una plataforma integral desarrollada para la gestión y control
de cumplimiento documental de contratistas en proyectos de construcción e
industria.

3.2 CARACTERÍSTICAS TÉCNICAS
----------------------------

• Backend desarrollado en NestJS (Node.js) con TypeScript
• Frontend desarrollado en Vue.js 3 con Quasar Framework
• Base de datos PostgreSQL 14
• Integración con Amazon S3 para almacenamiento de archivos
• Sistema de notificaciones por correo electrónico
• API RESTful con 183+ endpoints
• Autenticación JWT con control de roles (RBAC)
• Soporte para múltiples usuarios concurrentes

3.3 FUNCIONALIDADES DESTACADAS
------------------------------

• Dashboard de cumplimiento con métricas en tiempo real
• Control de vigencia de documentos con alertas automáticas
• Sistema de auditoría completo con historial de cambios
• Exportación de reportes a Excel
• Formularios dinámicos personalizables
• URLs presignadas para descarga segura de archivos
• Cierre de reportes ILV mediante enlace público
• Estadísticas y KPIs por módulo
